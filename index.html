Easy To Install

Download Docker Image Helm From Source

The simple way: grab the latest binary from the releases page and just run it with the sample configuration file :

./traefik --configFile=​traefik.toml

Using the official docker image:

docker run -d -p 8080:8080 -p 80:80 -v $PWD/​traefik.toml:​/etc/​traefik/​traefik.toml traefik

Install using Helm:

helm install stable/traefik

Clone the official repository from github:

git clone git@github.com:​containous/​traefik.git

And then follow the how to compile instructions from the contributing documentation.
